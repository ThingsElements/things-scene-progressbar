## node package를 설치한다.
```$ npm install```

## bower package를 설치한다.
```$ bower install```

## watchify를 실행한다. (소스를 변경하면, 자동으로 things-scene-gauge.js 파일을 리빌드한다.)
```$ npm run watch```

## 테스트 서버를 실행한다.
```$ polyserve```

## 브라우저에서 확인한다.
http://localhost:8080/components/things-scene-gauge/demo/index.html
